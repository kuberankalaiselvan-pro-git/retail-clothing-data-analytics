# Project Documentation

This folder contains detailed supporting documentation for the Retail Clothing Data Analytics project.

The document included here provides deeper coverage of:
- Data preparation and cleaning rationale
- Dimensional modeling decisions
- Power BI report structure
- DAX measure logic
- Statistical interpretation of findings
- Data-driven business recommendations

These materials were prepared as part of an academic submission and are included as **reference documentation**.

The main project summary, visuals, and key insights are presented in the root README.
