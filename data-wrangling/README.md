# Data Wrangling & Transformation

This folder documents the structured data cleaning and transformation process applied before analysis.

The transformations were performed using **Power Query** in Power BI and logged step‑by‑step to ensure transparency, auditability, and reproducibility.

## Key Objectives
- Standardize data types and formats
- Improve data quality and consistency
- Engineer analytical features
- Prepare fact and dimension tables for star schema modeling

The detailed transformation log is stored in the Excel file within this folder.
