# DAX Measures & Calculations

This section documents the key DAX measures used across the Power BI reports.

Measures were designed to ensure metric consistency, support time‑intelligence analysis, and enable reliable business insights.

## Measure Categories

- Core KPIs (Sales, Quantity, Orders)
- Profitability metrics (Gross Margin, Margin %)
- Time‑intelligence (YTD, MTD, QTD, YoY)
- Customer metrics (AOV, Retention, CLV)
- Diagnostic and data‑quality measures
- Dynamic measures for comparative analysis

Detailed measure definitions and examples will be documented here.
